# Author — 隐私政策

**最后更新：2026年4月23日**

## 1. 引言

本隐私政策说明了 Author 项目（以下简称"我们"）在您使用云同步服务和官方网页版时如何处理您的数据。我们致力于尊重您的隐私，并对我们的数据实践保持透明。

**简而言之：我们不会阅读您的创作内容，不会出售您的数据，仅收集提供同步服务和基础网页访问统计所必需的信息。**

## 2. 我们收集的数据

### 2.1 账户信息
当您创建账户时，我们会收集：
- **电子邮箱地址**（用于邮箱/密码注册）
- **Google 账户资料**（姓名和邮箱，用于 Google 登录）
- **用户 ID**（由 Firebase Authentication 自动生成）

### 2.2 同步内容
当您使用云同步功能时，以下数据将被存储：
- 您的创作作品（小说、章节、大纲、世界观设定等）
- 应用程序设置和偏好
- AI 对话历史记录（如果已启用云同步）

### 2.3 网页访问统计
在官方网页版中，我们可能通过 Firebase Analytics 和 Vercel Analytics 收集基础的汇总访问统计，例如页面浏览量、粗略的设备/浏览器信息和来源信息。我们不会用这些统计读取您的创作内容、API Key、AI 对话或同步文档。

### 2.4 我们**不**收集的数据
- 设备硬件标识符
- IP 地址日志（超出 Firebase 自动处理的范围）
- 与汇总访问统计无关的详细行为追踪数据
- 按键或输入遥测数据

## 3. 数据用途

您的数据**仅**用于：
- 在您的设备之间提供云同步服务
- 验证您的身份以保护您的账户
- 执行我们的服务条款（仅在严重违规情况下）

我们**不会**将您的数据用于：
- 训练 AI 模型
- 广告或营销
- 出售给第三方
- 针对个人的用户画像或广告导向行为分析

## 4. 数据存储与安全

- 所有同步数据存储在 **Google Cloud Firestore**（Google Cloud Platform 基础设施）上。
- 数据在传输过程中（TLS）和静态存储时（Google 托管加密）均已加密。
- 您的数据通过用户 ID 进行隔离——其他用户无法读取或写入您的数据。
- 我们使用 Firebase 安全规则在数据库层面强制执行严格的数据隔离。

## 5. 数据访问与调查权

- **我们不会主动、例行或系统性地阅读或审查您的创作内容。** 您的作品是您的私有财产，隐私是我们的默认立场。
- 我们可能**仅在**以下情况下访问或审查特定账户数据：
  1. **法律义务**：当法律、有效的法院命令或具有约束力的政府指令要求时
  2. **用户举报**：当我们收到来自其他用户或第三方的可信举报，指控特定内容违反了我们《服务条款》第 5 节（禁止内容）时。在此类情况下，我们保留对**仅限于被举报内容**进行**针对性审查**的权利，其唯一目的是确定是否发生了违规行为
  3. **自动系统警报**：如果我们未来部署自动化内容安全措施（例如结构模式检测、已知非法材料数据库的哈希匹配），且该系统标记了特定内容，我们保留对**仅被标记的内容**进行**针对性人工审查**的权利，以验证警报并采取适当的执法行动
- **范围限制**：上述第 2、3 种情况下的任何数据访问将：
  - **针对性**：仅限于调查中的特定内容——我们不会浏览、搜索或审查无关作品
  - **目的绑定**：仅为确定是否符合我们的服务条款而进行
  - **最小化**：由必要的最少数量的授权人员执行
  - **记录在案**：内部记录以确保问责
- 如果调查确认存在零容忍违规行为（如我们《服务条款》第 5.1–5.2 节所定义），我们可能会保留相关证据并根据适用法律的要求与执法机构共享

## 6. 第三方服务

Author 应用可能与以下第三方服务交互：
- **Firebase Authentication**（Google）— 用于账户管理
- **Cloud Firestore**（Google）— 用于数据存储
- **Firebase Analytics / Vercel Analytics** — 用于官方网页版的汇总访问统计
- **AI API 提供商**（OpenAI、Anthropic、Google 等）— 用于 AI 驱动的写作功能

当您使用 AI 功能时，您的提示词和上下文将根据**其**隐私政策发送给所选 AI 提供商。我们建议您查阅您所使用的 AI 提供商的隐私政策。

如果您使用自己的 API 密钥，AI 请求将直接从您的设备发送给提供商——我们无法访问这些请求，也不会记录日志。

## 7. 数据保留

- 在您的账户处于活跃状态期间，您的数据将被保留。
- 如果您删除账户，所有关联数据将从我们的系统中永久删除。
- 除 Firebase 的标准运营基础设施外，我们不维护单独的用户数据备份。

## 8. 您的权利

您有权：
- **访问**您的数据（所有同步数据均可在应用程序内查看）
- **导出**您的数据（Author 支持导出为多种格式）
- **删除**您的数据（通过删除账户或在应用内手动删除）
- **退出**云同步（Author 支持完全离线使用本地存储）

## 9. 自托管实例

如果您使用自己的基础设施自行托管 Author，本隐私政策不适用。自托管运营者应对自己的数据处理实践和隐私政策负责。

## 10. 儿童隐私

Author 的云同步服务不面向 13 岁以下的儿童。我们不会有意收集 13 岁以下儿童的个人信息。如果我们发现无意中收集了此类信息，将立即删除。

## 11. 政策变更

我们可能会不时更新本隐私政策。变更将反映在上方的"最后更新"日期中。在变更后继续使用云同步服务即表示接受更新后的政策。

## 12. 联系方式

如果您对本隐私政策有任何疑问，请在 [Author GitHub 仓库](https://github.com/YuanShiJiLoong/author) 上提交 issue。
